

<p><h1>Comments</h1></p>

<p>Comments are necessary when it comes to long and complex code.<br>

It helps others and yourself understand what the function of the code and it's purpose.</p>

<p><h2>Comment Syntax,</h2></p>

<p>One-line comments start with two forward slash characters.</p>

```
// This is a comment line
```

<p>Multi-line comments start with a forward slash and an asterisk and end with an asterisk and a forward slash.</p>

```
/* This is an example of two messages
I am a multiline comment
*/
alert('Learn');
alert('JS');
```

<p>If you need to disable code you can also insert it into a comment portion.</p>

**<p>Because im using a mac, if you highlight code or text and do command+/ a hot-key will automatically do it for you</p>**

**<p>Shift+Option+A for multi-line comments</p>**
