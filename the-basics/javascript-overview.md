

<p><h1>What kind of language is JS?</h1></p>

<p>JS is a interpreted language, meaning the code is executed directly in the browser without prior compliation.</p>

<p><h1>What does JS look like?</h1></p>

```
let x = 5;
let y = 10;
let result = x + y;
console.log("The result is: " + result);
￼VM454:4 The result is: 15
```

<p>You can run the above script directly in your Google Chrome browser, Ctrl+Shift+I (Inspect), click on console, paste, enter.</p>
