
<p><h1>What are variables?</h1></p>

<p>Variables are containers that store data values inside of them.</p>

<p>In JS like other languages, variables have a name.</p>

<p><h1>Declaring Variables</h1></p>

<p>Variables can be declared in three ways,</p>

> - var - Function-Scoped
> - const - Block-Scoped
> - let - Block-Scoped

<p><h1>Data Types</h1></p>

<p>Number, String, Boolean, Null, Undefined, and Object for more complex data.</p>

<p><h1>Functions</h1></p>

<p>A function represents a block of code designed to perform a specific task.</p>

<p><h1>Loops</h1></p>

<p>For, While, Do..While (all obviously lowercase)</p>

<p><h1>What is the Request-Response Cycle?</h1></p>

<p>When a users browser(client) sends a request to a web-server and the server responds with the information.</p>
